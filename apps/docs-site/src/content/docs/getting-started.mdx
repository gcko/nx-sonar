A [Nx](https://nx.dev/) Plugin that contains executors and generators to support scanning projects using [Sonar](https://www.sonarsource.com/products/sonarqube/)

## Installation

<Tabs>
<TabItem label="npm">

```sh
npm install @gcko/nx-sonar --save-dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm add -D @gcko/nx-sonar
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn add @gcko/nx-sonar -D
```

</TabItem>
</Tabs>
